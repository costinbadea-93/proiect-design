Romsa Security Systems

Romsa Security Systems a fost infiintata in 1997 si este specializata in distributia in Romania a unor sisteme de securitate produse in Africa de Sud. Industria sistemelor de securitate din Africa de Sud ofera o gama variata de produse, la tehnologie de varf, inalta calitate si preturi competitive.Romsa Security Systems este reprezentata in Romania de catre Romsa Access Systems S.R.L.


Fabricantii si gamele de produse oferite includ:

  • Sisteme de control acces - Impro
  • Software de pontaj - Jarrison Time
  • Cititoare de amprenta si sisteme biometrice - Nitgen
  • Detectoare vehicule pentru parcari si controlul traficului rutier - Nortech


    • Prin cele doua firme Romsa Security Systems in Africa de Sud si Romsa Access Systems in Romania, RSA si RO sunt pe acelasi fus orar, ne situam aproape atat de fabricantii de sisteme cat si de clientii nostri din tara. Aceasta pozitionare asigura clientilor nostri o garantie in gasirea celor mai potrivite solutii, informatie de ultima ora despre produse si rezolvarea operativa a oricaror probleme aparute pe teren.


    Misiunea Romsa este de a oferi cu profesionalism si integritate produse de inalta tehnicitate si calitate, dublate de un excelent suport tehnic.
